### Production Deployment Commands

```powershell
# First time on a new machine:
cd C:\path\to\ims-pro
setup.bat 192.168.1.100    # replace with your server LAN IP

# Rebuild after code changes:
docker compose up -d --build

# Check health:
docker compose ps
docker compose logs --tail=50 api

# Restart a single service:
docker compose restart api
```

### Access URLs (Production via Nginx)
- LAN: `http://SERVER_IP` (port 80, routed through Nginx)
- Direct API: `http://SERVER_IP:4000`
- Direct Web: `http://SERVER_IP:3000`
